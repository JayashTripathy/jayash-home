## Lets start with logic part

### Select your homepage container

- First using the `querySelector` method, select the container that you want to add the back-to-top functionality to.
  For this example, it is a `document`

```jsx:app.js
    let viewPortHeight = document.documentElement.clientHeight;
    let scrolledCurrent = viewPortHeight + document.documentElement.scrollTop;
    let totalHeight = document.documentElement.scrollHeight;
```
